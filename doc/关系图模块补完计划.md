# 关系图模块补完计划

> 创建日期：2026-07-16
> 依据：`doc/关系图模块改造计划.md`、`doc/关系图模块实施计划.md`
> 状态：已完成

## 一、当前差距

1. 样式层仍有两段 `.graph-*` 区块，新旧关系图样式混在不同位置，容易互相覆盖。
2. `applyGraphFilters` 在过滤结果为空时会静默回退原图，用户无法知道是筛选条件导致无结果。
3. 伏笔模式缺少“显示已回收 resolved 伏笔”的开关。
4. 力导向视图缺少位置缓存与 `dataKey` 控制，过滤/切换时容易重复跑 ForceAtlas2。
5. MiniMap 尚未实现。
6. `buildGraphDocument` 已存在，但 UI 没有 Markdown 文档复制/导出入口。
7. 洞察卡片不能点击聚焦对应节点。
8. `relationshipStore.ts` 的边权重仍偏简化，未显式实现“五信号边权重算法”。
9. 重复边合并还不完整，边描述没有规范合并，部分边 key 判断未统一归一化。
10. `RelationshipGraph.tsx` 子组件已有雏形，但右侧栏和右键菜单仍混在主 JSX 中，维护性不足。

## 二、执行顺序

### Phase 1：样式分区维护

- 保留一组共享图谱样式，供 `ChapterGraph` 继续复用：`.graph-panel`、`.graph-canvas-area`、`.graph-svg`、`.graph-link-tooltip`、`.graph-legend-line` 等。
- 新关系图模块使用 `relationship-graph` 根 class 做作用域，专属样式写成 `.relationship-graph .graph-*`。
- 避免旧共享样式与新关系图专属样式使用同级同名 selector 互相覆盖。

### Phase 2：过滤与伏笔模式

- 修改 `applyGraphFilters`，过滤为空时返回空结果和原因，不再静默恢复原图。
- 在关系图 UI 中显示“当前筛选无节点”，保留“一键重置筛选”。
- 伏笔模式默认隐藏 `resolved`，增加“显示已回收”开关。

### Phase 3：布局与视图补完

- 给力导向图增加位置缓存。
- 使用 `dataKey` 控制 graphology/Sigma 图实例重建，数据未变时不重复布局。
- 增加轻量 MiniMap。
- 文档视图增加 Markdown 复制和导出。

### Phase 4：交互补完

- 洞察卡片点击后聚焦相关节点。
- 抽出 `GraphSidebar` 与 `NodeContextMenu`，降低主组件 JSX 复杂度。

### Phase 5：边权重与合并规则

- 在 `relationshipStore.ts` 显式实现五信号边权重：
  - 直接链接
  - 同源共现
  - 共同邻居
  - 类型亲和
  - 关系强度
- 统一无向边 key 归一化。
- 合并重复边时累加 mentions、合并 first/last、保留更强显式关系、合并描述。

### Phase 6：验证

- 运行 TypeScript 检查。
- 运行构建。
- 检查共享图谱样式与 `RelationshipGraph` 专属样式分区清晰，没有同级同名 selector 互相覆盖。
- 检查 codegraph 状态并同步索引。

## 三、完成记录

- 已将共享图谱样式与 `RelationshipGraph` 专属样式分区维护：共享 `.graph-*` 只保留 `ChapterGraph` 可复用基础样式，新关系图样式统一挂在 `.relationship-graph .graph-*` 下。
- 已取消过滤为空时静默恢复原图，改为返回空结果并由 UI 显示“当前筛选无节点”。
- 已增加伏笔模式“显示已回收伏笔”开关，默认隐藏 `resolved`。
- 已增加 ForceAtlas2 位置缓存、`dataKey` 控制和轻量 MiniMap。
- 已给文档视图增加 Markdown 复制与导出入口。
- 已让洞察卡片可点击聚焦图谱节点。
- 已抽出 `NodeContextMenu`、`GraphToolbar`、`GraphLegend`、`NodeDetail`、`InsightsPanel` 等子组件。
- 已在数据层统一无向边 key，合并重复边描述，并按直接链接、同源共现、共同邻居、类型亲和、关系强度五个信号重新计算边权重。
- 已通过 `npm exec tsc -- --noEmit`。
- 已通过 `npm run build`。
- 已执行 `codegraph sync`，`codegraph status` 显示索引最新。
